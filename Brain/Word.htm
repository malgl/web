ㄱ : 어금닛소리, 혀뿌리가 목구멍을 닫는 꼴 ㄱ → ㅋ (ㆁ)
ㄴ : 혓소리, 혀가 윗잇몸에 붙는 꼴 ㄴ → ㄷ → ㅌ (ㄹ[半혓소리])
ㅁ : 입술소리, 입의 꼴 ㅁ → ㅂ → ㅍ
ㅅ : 잇소리, 이의 꼴 ㅅ → ㅈ → ㅊ (ㅿ[半잇소리])
ㅇ : 목구멍소리, 목구멍의 꼴 ㅇ → ㆆ → ㅎ
