단축명령

  • Layer복사 : ctrl + j
  • 흑백변환 : ctrl+shift+u
  • Invert : ctrl + i
  • Layer 합치기 : ctrl + e

일반적인 사진에 색연필로 그린듯한 효과 주기

  1. 우선 사진을 불러온후 Layer 복사후 흑백변환을 합니다.
  2. 흑백변환한 Layer를 복사한 후 색상반전(Invert)을 합니다.
  3. Color Dodge 를 해줍니다.
  4. 그후에 Filter -> Other -> Minimum 을 해줍니다. (값 : 1)
  5. 그후 Layer -> Layer Style -> Blending Options 을 해줍니다. 작은삼각형 (원래는 하나로 합쳐져 있음)을 Alt key를 누른상태로 움직여 적당하다 생각되는 정도까지 움직여 줍니다. Alt key를 누르며 움직이면 삼각형이 반으로 쪼개지면서 움직이게 됩니다.
  6. 흑백으로 된 2개의 Layer를 합친후 원본Layer를 복사후 위로 배치합니다.
  7. Color로 변경
  8. 원본Layer를 제외한 나머지 Layer를 합친후 합친 Layer를 다시 복사합니다. 그후 Filter -> Brush Strokes -> Angled Strokes 를 적용하고 Fill을 30~40 정도로 맞춰줍니다.
  9. Filter 적용 Layer를 합친후 다시 Layer 복사를 한후 Filter -> Texture -> Texturizer 를 적용하고 Fill을 10~20 정도로 맞춰줍니다.
  10. Filter 적용 Layer를 합친후 다시 Layer 복사를 한후 Filter -> Blur -> Radial Blur 를 적용하고 Fill을 5 정도로 맞춰줍니다.
  11. Filter 적용 Layer를 합친 후 Image -> Adjustments -> Shadow/Highlight 를 써서 적당하다고 판단될 정도로 값을 줍니다.

피부 보정

  1. Layer를 복사하고 Overaly 선택 후 Invert 적용
  2. Filter -> Other -> High Pass (Radius : 9, 적당한 값 선택)
  3. Filter -> Blur -> Gaussian Blur (Radius : 1, 적당한 값 선택)
  4. Alt + Add layer mask(Filter 효과 숨김)
  5. Brush로 피부를 문질러 준다.(mask의 문진 부분에 Filter 효과 나타남)
